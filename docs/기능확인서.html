ERP + CRM 통합 시스템

기능 확인서 (Feature Confirmation)
문서버전 : v1.1
작성일 : 2026. 03. 05
본 문서는 ERP + CRM 통합 관리 시스템의 개발 범위를 확정하기 위한 기능 확인서입니다.
각 항목을 확인하신 후 확인란에 체크해 주시기 바랍니다.
체크되지 않은 항목은 개발 범위에서 제외될 수 있습니다.
1
계정 및 권한 관리
로그인, 계정 관리, 역할별 권한 분리
No 기능 상세 내용 확인
1-1 로그인 / 로그아웃 아이디·비밀번호 입력 후 로그인, 세션 기반 인증 유지, 로그아웃 시 세션 파기 - 로그인 성공 시 대시보드로 이동
- 세션 유효시간 8시간 (근무시간 기준)
- 로그인/로그아웃 시간 자동 기록
1-2 직원 계정 생성 관리자가 직원 계정을 새로 생성 - 입력: 아이디, 비밀번호, 이름, 직급, 연락처, 이메일(선택)
- 아이디 중복 자동 검사
- 비밀번호 암호화 저장
1-3 직원 계정 수정 관리자가 기존 직원 정보를 수정 - 이름, 직급, 연락처, 이메일, 비밀번호 변경 가능
- 수정 이력 로그 기록
1-4 직원 계정 삭제 (비활성화) 관리자가 직원 계정을 비활성화 처리 - 완전 삭제가 아닌 비활성화 처리
- 비활성화 시 배정된 DB 자동 회수
- 기존 데이터(메모, 이력 등) 보존
1-5 역할별 권한 분리 관리자 / 직원 2개 역할, 기능별 접근 제한 관리자 전용 : 전체 데이터 조회, 직원 계정 관리, DB 배분/회수, 엑셀 다운로드, 전체 통계, 활동 로그, 업체 등록/수정

직원 : 본인 배정 DB만 조회/수정, 콜 처리, 메모 작성, 본인 매출/통계 조회, 엑셀 업로드 (다운로드 불가)
1-6 인센티브 % 설정 직원별 인센티브 비율을 설정하고 매출 등록 시 자동 계산 관리자 - 관리자가 직원별 인센티브 비율(%) 설정
- 매출 등록 시 설정된 인센티브 비율로 자동 계산
- 인센티브 금액 = 순마진 × 인센티브 비율(%)
- 인센티브 비율 변경 이력 기록
2
ERP - 매출 관리
거래 업체 등록, 매출/순마진 관리, 계약 이월
No 기능 상세 내용 확인
2-1 업체 등록 신규 거래 업체를 시스템에 등록 관리자 기본 항목 : 담당자, 등록일, 상품명, 상호명, 결제금액, 계산서 발행금액, 실행비, 직급

상세 항목 : 매출 등록일, 작업 시작일, 작업 종료일, 계약 시작일/종료일, 업체명, 대표자명, 연락처, 결제구분, 사업자번호, 작업 키워드, 작업 내용, 이메일, 네이버 계정 정보, 실행비 입력란
2-2 업체 수정 등록된 업체 정보를 수정 관리자 - 모든 필드 수정 가능
- 금액 수정 시 순마진 자동 재계산
- 수정 전후 데이터 이력 보존
2-3 업체 목록 조회 업체 목록을 다양한 조건으로 조회 - 필터: 기간(년/월/일), 담당자, 결제구분
- 검색: 상호명, 대표자명, 연락처, 사업자번호 (부분 검색)
- 정렬: 등록일, 결제금액, 순마진 기준
- 표시 개수: 50 / 100 / 300 / 500 / 1000건 선택
- 관리자는 전체, 직원은 본인 담당만 조회
2-4 업체 상세 조회 업체의 전체 정보 및 히스토리 조회 - 기본정보 + 상세정보 전체 표시
- 일자별 메모 히스토리 목록
- 순마진 자동계산값 표시
2-5 매출 자동 계산 금액 입력 시 부가세·순마진 실시간 자동 계산 - 부가세 = 결제금액 / 11
- 순마진 = 결제금액 - 실행비 - 부가세
- 결제금액 또는 실행비 입력/수정 시 즉시 재계산
- 집계: 총 매출, 총 실행비, 총 부가세, 총 순마진
2-6 계약 이월 (연장) 기존 업체 정보를 복사하여 신규 계약으로 등록 관리자 - "이월" 버튼 클릭 시 기존 정보 자동 복사
- 복사 항목 : 업체명, 대표자명, 연락처, 사업자번호, 결제구분, 작업키워드, 작업내용, 이메일, 네이버 계정, 담당자, 결제금액, 실행비, 메모
- 초기화 항목 : 계약 시작일/종료일
- 이월된 결제금액, 실행비, 메모는 수정 가능
- 계약기간 수정 후 신규 레코드로 별도 저장
- 원본 데이터는 그대로 유지
2-7 일자별 메모 (히스토리 누적) 업체별 메모를 날짜순으로 누적 기록 - 메모 작성 시 작성자·작성시간 자동 기록
- 기존 메모 덮어쓰기 금지 (수정/삭제 불가)
- 새 메모만 추가 가능, 시간 역순(최신순) 표시
3
CRM - 네이버 쇼핑 DB 관리
엑셀 업로드, 콜 상태 관리, DB 배분/회수
No 기능 상세 내용 확인
3-1 엑셀 DB 업로드 엑셀 파일을 업로드하여 쇼핑 DB 등록 관리자+직원 - 지원 형식: .xlsx, .xls, .csv
- 전화번호 기준 중복 자동 체크
- 중복 시 표시 후 선택 (건너뛰기/덮어쓰기)
- 업로드한 직원 계정에 자동 귀속
- 업로드 결과 표시 (전체/성공/중복 건수)
3-2 업로드 이력 관리 엑셀 업로드 이력을 기록하고 조회 - 업로드 일시, 파일명, 전체/중복/성공 건수, 업로드자 기록
- 관리자: 전체 이력 / 직원: 본인 이력만 조회
3-3 쇼핑 DB 목록 조회 필터/검색 기능으로 쇼핑 DB 목록 조회 - 필터: 상태값, 기간(년/월/일), 담당자(관리자만)
- 검색: 상호명, 연락처, 이름 (부분 검색)
- 정렬: 등록일, 상태, 최근 활동일 기준
- 표시 개수: 50 / 100 / 300 / 500 / 1000건
- 관리자는 전체, 직원은 본인 배정만 조회
3-4 콜 상태 변경 DB 레코드의 콜 처리 상태를 변경 관리자+직원 - 상태값: 부재 / 재통 / 가망 / 계약완료 / 안함
- 상태 변경 시 변경 시간 자동 기록
- 변경자 자동 기록
- 상태 변경 히스토리 누적 저장 (이전 기록 보존)
3-5 DB 배분 관리자가 쇼핑 DB를 직원에게 배분 관리자 - 다건 선택 후 일괄 배분 가능
- 배분 대상 직원 선택
- 이미 배분된 DB는 회수 후 재배분 필요
- 배분 이력 자동 기록
3-6 DB 회수 관리자가 배분된 DB를 회수 관리자 - 다건 선택 후 일괄 회수 가능
- 회수 시 담당자 해제 (미배분 상태)
- 기존 상태 히스토리·메모는 유지
- 회수 이력 자동 기록
3-7 메모 작성 쇼핑 DB 레코드별 메모 누적 기록 - 작성자·작성시간 자동 기록
- 덮어쓰기 금지, 히스토리 누적 방식
- 최신순 표시, 삭제 불가
4
CRM - 네이버 플레이스 DB 관리
외부 광고 매체 DB 등록, 콜 상태 관리, 배분/회수
No 기능 상세 내용 확인
4-1 플레이스 DB 등록 관리자가 외부 광고 매체 DB를 직접 등록 관리자 - 입력 항목: 상호명, 연락처, 지역, DB 등록일자, 알게된 경로, 메모
- 연락처 중복 체크 지원
4-2 플레이스 DB 목록 조회 필터/검색 기능으로 플레이스 DB 목록 조회 - 필터: 상태값, 기간(년/월/일), 담당자(관리자만), 지역, 알게된 경로
- 검색: 상호명, 연락처 (부분 검색)
- 표시 개수: 50 / 100 / 300 / 500 / 1000건
- 관리자는 전체, 직원은 본인 배정만 조회
4-3 콜 상태 변경 DB 레코드의 콜 처리 상태를 변경 관리자+직원 - 상태값: 부재 / 재통 / 가망 / 거절 / 계약완료
- 상태 변경 시 변경 시간·변경자 자동 기록
- 상태 변경 히스토리 누적 저장
4-4 DB 배분 관리자가 플레이스 DB를 직원에게 배분 관리자 - 다건 선택 후 일괄 배분 가능
- 배분 대상 직원 선택
- 배분 이력 자동 기록
4-5 DB 회수 관리자가 배분된 DB를 회수 관리자 - 다건 선택 후 일괄 회수 가능
- 기존 상태 히스토리·메모 유지
- 회수 이력 자동 기록
4-6 메모 작성 플레이스 DB 레코드별 메모 누적 기록 - 작성자·작성시간 자동 기록
- 덮어쓰기 금지, 히스토리 누적 방식
- 최신순 표시, 삭제 불가
5
대시보드
매출 현황, 위젯 카드, 그래프 시각화
No 기능 상세 내용 확인
5-1 관리자 매출 집계 대시보드 전체 매출 현황을 한눈에 확인 관리자 - 조회 단위: 년 / 월 / 일 선택
- 자동 집계: 총 매출, 총 실행비, 총 부가세, 총 순마진, 총 계산서 발행 금액
- 결제 금액 우측에 계산서 발행 금액 표시
- 직원별 매출·순마진 비교
- 목표 대비 달성률 표시
- 매출 등록일 / 작업 시작일 / 작업 종료일 기준으로 업체별 마감 일자 표시
- 기간 선택 필터
- 엑셀 다운로드 가능
5-2 직원 개인 매출 대시보드 본인 매출 현황만 조회 직원 - 조회 단위: 년 / 월 / 일
- 표시: 본인 총 매출, 총 실행비, 총 부가세, 총 순마진, 총 계산서 발행 금액
- 결제 금액 우측에 계산서 발행 금액 표시
- 매출 등록일 / 작업 시작일 / 작업 종료일 기준으로 업체별 마감 일자 표시
- 월별 그래프 표시
- 목표 대비 달성률 표시
- 인센티브 기준 자동 계산 (옵션)
- 타 직원 데이터 접근 불가, 다운로드 불가
5-3 위젯형 요약 카드 대시보드 상단에 주요 지표를 카드 형태로 표시 - 금일 매출 / 금월 매출
- 금일 순마진 / 금월 순마진
- 금일 신규 DB 건수
- 금일 계약완료 건수
- 페이지 로드 시 자동 갱신
5-4 그래프 시각화 매출·실적 데이터를 그래프로 표현 - 월별 매출 추이 (선 그래프)
- 직원별 실적 비교 (막대 그래프)
- 상태별 DB 현황 (파이 차트)
6
통계
계약 건수 집계, 상태별 통계, 매출 추이, 직원 비교
No 기능 상세 내용 확인
6-1 직원별 계약완료 건수 집계 직원별 계약 성사 건수 자동 집계 - 쇼핑 DB + 플레이스 DB 합산 집계
- 조회 단위: 년 / 월 / 일
- 관리자: 전체 직원 / 직원: 본인만
6-2 상태별 통계 그래프 DB 상태별 분포를 그래프로 시각화 관리자 - 파이 차트 + 막대 그래프
- 쇼핑 DB / 플레이스 DB 각각 + 통합 표시
6-3 기간별 매출 통계 기간별 매출 추이를 그래프로 시각화 - 선 그래프 + 막대 그래프
- 조회 단위: 일별 / 월별 / 년별
- 표시 항목: 매출, 실행비, 부가세, 순마진, 계산서 발행 금액
- 관리자: 전체 / 직원: 본인만
6-4 직원별 실적 비교 직원 간 실적을 비교 차트로 표시 관리자 - 비교 항목: 매출액, 순마진, 계약완료 건수, 콜 건수
- 수평 막대 그래프
7
공통 기능
메모, 검색/필터, 활동 로그, 엑셀 다운로드
No 기능 상세 내용 확인
7-1 메모 기능 (전체 공통) 모든 DB/업체에 적용되는 메모 기능 - DB별 개별 메모 작성 가능
- 작성자 및 작성시간 자동 기록
- 덮어쓰기 금지 (히스토리 누적 방식)
- 시간 역순(최신순) 표시
- 메모 수정/삭제 불가
7-2 조회 / 필터 / 검색 (전체 공통) 모든 목록 화면에 적용되는 공통 기능 - 기간 필터: 년 / 월 / 일 선택
- 표시 개수: 50 / 100 / 300 / 500 / 1000건 선택
- 검색: 이름, 전화번호, 상호명 (부분 검색 지원)
